# LuxeMart – E-Commerce Store

## Project Description
LuxeMart is a modern front-end e-commerce website developed using HTML, CSS, and JavaScript. The website provides users with an interactive online shopping experience where they can browse products from different categories, add products to favorites and cart, apply filters, and complete a simulated payment process.

The project focuses on implementing the core functionalities of an online shopping platform while maintaining a clean and responsive user interface.

## Features Implemented

### User Authentication
* Login system
* Signup system
* Validation for user inputs
* Local storage based authentication

### Product Management
* Multiple product categories
* Dynamic product display
* Product cards with detailed information
* Product filtering and sorting

### Shopping Features
* Add to cart
* Favourite products system
* Cart total calculation
* GST and delivery charge calculation
* Payment simulation
* Receipt generation

### UI/UX Features
* Responsive design
* Modern glassmorphism styling
* Interactive buttons and cards
* Smooth navigation between pages

## Technologies Used

| Technology | Purpose |
| :--- | :--- |
| **HTML5** | Structure of webpages |
| **CSS3** | Styling and responsiveness |
| **JavaScript** | Website functionality and interactivity |
| **Local Storage** | Storing user, cart, and favourite data |
| **Session Storage** | Temporary payment and receipt data |

## Additional Features
* Dynamic product rendering
* Category-based product pages
* Filter by brand, price, and rating
* Cart persistence using localStorage
* Downloadable receipt generation
* Simulated payment timer

## How to Run the Project
1. **Download or Clone:** Download the ZIP file or clone the repository using `git clone`.
2. **Open Folder:** Navigate to the project directory on your local machine.
3. **Launch:** Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge).
4. **Access:** Sign up or log in using the authentication system to access the full features of the store.

## Future Improvements
* **Backend Integration:** Moving from local storage to a live server environment.
